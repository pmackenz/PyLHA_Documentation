UI.HazardParser.HazardParser : public object

Parses a folder for files containing Hazard Information and Data.

Use this class to locate and extract human readable information from previously saved hazard information files.

By default, any instance of this class searches in ‘Documents/PyLHA/ExistingHazardData’. A different search folder can be specified using setPath(FULL_PATH_TO_FOLDER)
