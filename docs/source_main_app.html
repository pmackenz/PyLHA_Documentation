qPLHA application

How to launch the application?

  • starting the app using the graphical user interface (GUI)

    MacOS or linux: 

    $ python3 qPLHA.py
    Windows systems: 
    > python3.exe qPLHA.py

  • starting the app in batch mode (command line mode)

    MacOS or linux: 

    $ python3 qPLHA.py --batch
    Windows systems: 
    > python3.exe qPLHA.py --batch

  • starting the app with precomputed data

    If results data from a previous batch-run has been exported to a file FILE_NAME.xr, the app can be launched and results can be viewed without recomputing using the following option:

    MacOS or linux: 

    $ python3 qPLHA.py --data=FILE_NAME.xr
    Windows systems: 
    > python3.exe qPLHA.py --data=FILE_NAME.xr
